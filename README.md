A final exam task at CodeAcademy bootcamp. The goal was to create small functional website for a gym, with strict rules of how to style nearly everything and what to use. Website was written in PHP, using OOP, MVC design pattern and connecting it to JavaScript API. 

**For deployed version visit: [page-php-oop-api.herokuapp.com](https://page-php-oop-api.herokuapp.com/)** 

### Tech Stack

The following technologies were used in the project:
* [PHP](https://www.php.net/) - a popular general-purpose scripting language that is especially suited to web development.
* [Composer](https://getcomposer.org) - a Dependency Manager for PHP.
* [Travis CI](https://travis-ci.com/) - continuous integration service used to build and test project.
* [Heroku](https://www.heroku.com) - Cloud Application Platform for hosting.

### Running project

**Visit: [page-php-oop-api.herokuapp.com](https://page-php-oop-api.herokuapp.com/)**

**OR**

Run project locally:

1. Clone GitHub repo for this project locally
    ```bash
    git clone https://github.com/PauliusU/gym_page_API_OOP.git
    ```
2. cd into project
    ```bash
    cd gym_page_API_OOP
    ```
   _**Note**: Make sure that web root folder is within "public_html"._
3. Install Composer Dependencies
    ```bash
    composer install
    ```
   _**Note**: If you get errors during install, please clear your cookies in browser_
4. [Optional] install sample data (sample comments in feedback page and users)
    
    Launch in a browser:
    ``` bash
    http://yourdomain.lt/install
    ```
5. Visit home page in a browser
    ``` bash
    http://yourdomain.lt/
    ```
